# Claude RTL Extension 🇮🇱

> Chrome extension that automatically fixes Hebrew text alignment on claude.ai

## The Problem

claude.ai doesn't support RTL (right-to-left) text direction for Hebrew speakers.
All responses appear left-aligned, making Hebrew text awkward to read.

## The Solution

This extension automatically detects Hebrew text and applies RTL alignment to:
- ✅ Claude's responses
- ✅ Bullet points and numbered lists
- ✅ User messages in chat
- ✅ The input box
- ✅ Code blocks remain LTR (untouched)

## Installation

1. Download or clone this repository
2. Open Chrome → `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** → select the folder
5. Done ✓
